  • Website: Relax in the Lounge
    https://www.relaxinthelounge.com
    Objective
    The objective of this website is to allow users easily access various media contents like Television, Radio, News, Magazines, Podcasts etc. from one source.
    Technologies Used
    This website, which is a Single Page Application, is developed using HTML5, SCSS (SASS), JavaScript and AngularJS 1.6 on the front-end and LAMP stack (Linux, Apache, MySQL, PHP) on the back-end and. Data is transferred in JSON format.
    Responsive Design
    This website adapts to any device screen sizes, from desktop to mobile phones.
    User Experience
    This website is designed for the best User Experience. Users can easily register from the home page itself without navigating to another page. When a category is clicked, its content is displayed without refreshing the entire page. The tile based designed help users easily access the content with reduced scrolling. Navigation menus are positioned at the bottom left and right corner for easy access from mobile devices. The menus does not scroll even when the content scrolls for quick navigation to different categories or sub-items.
    Bookmark Feature
    Users can bookmark their favorite items easily and navigates quickly to the favorited items page. If registered, users can access their favorites from any systems they log in.
    Sort
    Contents can be easily sorted either by the content names or by the number of user views. When a user view a content, the user view count is logged into the database. This is used for sorting the items. The most viewed item comes on the top by default. The user can also sort the items against the names.
    Sleep Timer
    For radio section, sleep timer feature is incorporated so that the users can specify a time to stop the radio.
